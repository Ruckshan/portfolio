What are Bootstrap Images?

Bootstrap Images are the content element that allows you to opt images into responsive behavior (so they never become larger than their parent elements) and add lightweight styles to them‚Äîall via classes.

Installation

Manual installation (zip package)

To take advantage of our Bootstrap images component and use them in your project, you first need to install the MDB 5 Free package

MDB CLI

Watch our Quick Start Tutorial to discover and use the full potential of MDB 5 and MDB CLI

NPM

Prerequisites

Before starting the project make sure to install Node LTS (12.x.x recommended).

Installation

To install MDB UI KIT in your project easily type the following command in the terminal: npm i mdb-ui-kit

Importing JS modules

You can import the entire library or just individual modules:

1import { Input } from 'mdb-ui-kit'; // module ```

Customization

Responsive images

Images in MDB are made responsive with .img-fluid. This applies max-width: 100%; and height: auto; to the image so that it scales with the parent element.

1<img src="https://mdbootstrap.com/img/new/slides/041.jpg" class="img-fluid" alt="..." />
Thumbnails

In addition to our border-radius utilities, you can use .img-thumbnail to give an image a rounded 1px border appearance.

1<img
2  src="https://mdbootstrap.com/img/new/standard/city/041.jpg"
3  class="img-thumbnail"
4  alt="..."
5/>
Masks

You can cover the image with mask to achieve the desired contrast and for example, place text on it.

1<div class="bg-image">
2  <img
3    src="https://mdbootstrap.com/img/new/standard/city/053.jpg"
4    class="img-fluid"
5    alt="Sample"
6  />
7  <div class="mask" style="background-color: rgba(0, 0, 0, 0.6)">
8    <div class="d-flex justify-content-center align-items-center h-100">
9      <p class="text-white mb-0">Can you see me?</p>
10    </div>
11  </div>
12</div>
